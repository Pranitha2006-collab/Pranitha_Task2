# Guess The Number Game

## Description

A Java-based Number Guessing Game where the computer generates a random number between 1 and 100 and the player tries to guess it.

## Features

* Random number generation
* Higher/Lower hints
* Multiple rounds
* Limited attempts
* Score calculation
* Java Swing dialog boxes

## Technologies Used

* Java
* Swing (JOptionPane)

## How to Run

Compile:
javac GuessTheNumberGame.java

Run:
java GuessTheNumberGame
