# 🎯 Number Guessing Game

## 📖 Introduction

The Number Guessing Game is a Java-based console application that provides an interactive and engaging gaming experience. The system generates a random number within a specified range, and the player must guess the correct number using logical thinking and hints provided by the application.

This project was developed as part of the **Oasis Infobyte Java Development Internship** to demonstrate the use of Java programming concepts such as loops, conditional statements, random number generation, and user interaction.

---

## ✨ Features

* 🎲 Random number generation
* 🎮 Interactive user gameplay
* 📈 Hint system (Too High / Too Low)
* 🔄 Multiple attempts for guessing
* 🏆 Score tracking based on performance
* 🔁 Play Again functionality
* ✅ User-friendly console interface

---

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)
* VS Code
* Git & GitHub

---

## 🚀 How It Works

1. The application generates a random number.
2. The player enters a guess.
3. The system compares the guess with the generated number.
4. Appropriate hints are displayed:

   * Too High
   * Too Low
5. The process continues until the correct number is guessed.
6. The final score is displayed.
7. The player can choose to play another round.
